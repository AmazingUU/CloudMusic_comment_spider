# CloudMusic_comment_spider
网易云音乐评论爬虫

# 项目博客

# 简介
爬取网易云音乐排行榜中云音乐飙升榜的歌曲以及对应的热门评论和前20最新评论。

# 功能

 1. 爬取云音乐飙升榜中的歌曲相关信息
 2. 爬取对应歌曲的热门评论
 3. 爬取对应歌曲的前20条最新评论
